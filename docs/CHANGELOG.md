# Changelog

## [1.0.0] - 2026-07-22

### Added
- Initial release of Second Brain
- Rich text note editor (flutter_quill)
- Document import (PDF, DOC, DOCX, PPT, TXT, XLS, XLSX)
- Image import from gallery and camera capture
- Voice recording and playback
- Bookmark saving with auto-metadata fetch
- Collections for organization
- Global tags system with multi-tag support
- Universal favorites (notes, docs, images, voice, bookmarks)
- Global search across all content
- Dark, Light, System theme modes
- Material 3 design
- Google Keep inspired card UI
- Expandable FAB with animation
- Animated bottom navigation
- Responsive layout (phones, tablets)
- Offline-first (Drift SQLite)
- go_router navigation
- Riverpod state management
- Freezed domain models

### Technical
- Feature-first clean architecture
- Repository pattern
- Drift database with DAOs
- Riverpod providers with code generation
- build_runner for code generation
